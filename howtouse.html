How to Use

Welcome to Ergon! Follow these steps to get started:

  1. Click on the "Tasks" option to add new tasks.
  2. Set a due date and time for each task using the respective input fields.
  3. Set a priority option (Rank) for each task using the respective input field. The task's Rank will be displayed as an arrow for your convenience.
  4. Add a new task using the pencil button(✏️).
  5. Manage your tasks by marking them as completed or deleting them.
  6. Change the theme of yor task container using the settings button (⚙️).

Enjoy organizing your tasks efficiently!
